# Mtkiao-Bot
一個很普通的Discord Bot  
Code 寫得很爛w 勿噴  
discord.js版本還停在13.x 之後應該會升級到14.x  

## Setup
1. Config/config.json裡面的東西需要改成自己的
2. .env寫入Bot Token
3. 安裝node環境(16版本以上)
4. 在Bot文件夾裡打開終端輸入: `npm install`
5. 更新Bot指令: `node deploy-commands.js`  
6. 運行Bot: `node inedx.js`
